About MindSpace Mood Tracker

MindSpace is a client-side mood tracking application that helps users record emotional state, note context, and review patterns over time.

Core Features

  • Private local data: Logs are saved in browser localStorage and stay on the user device.
  • Live analytics: The dashboard calculates total logs, average score, common mood, range, distribution, and recent trend.
  • Safer rendering: User notes are rendered as text so typed content does not become page markup.

How to Use

  1. Select the mood that best matches your current state.
  2. Add a short note about what influenced that feeling.
  3. Click "Log Mood" to save the entry locally.
  4. Review the history and analytics to understand your emotional pattern.
