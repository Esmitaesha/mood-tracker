About MindSpace Mood Tracker

MindSpace is a client-side utility application designed to help individuals monitor their emotional well-being over time. Tracking mood variations is an evidence-based practice in cognitive behavioral psychology that helps users identify environmental triggers and emotional trends.

Key Academic & Technical Features

  • Zero Server Footprint: Utilizes the HTML5 Web Storage API (localStorage) to store data completely locally, ensuring data privacy.
  • Dynamic DOM Manipulation: Re-renders tracking data, history states, and analytical breakdown instantly without page reloads.
  • Custom-Built Micro Data Analytics: Computes emotional averages and distribution trends programmatically via JavaScript arrays and renders them using purely mathematical flexbox scale mappings.

How to Use the App

  1. Select an Emoji: Click on the emoji matching your current emotional state on the main dashboard.
  2. Add Context: Type a brief note outlining what factors contributed to your mood (work, sleep, health, etc.).
  3. Log It: Click "Log Mood". Your log is instantaneously timestamped and safely written to your browser's local sandbox.
  4. Review & Analyze: Scroll down to view past history logs and study the distribution bar graph to observe your overall mental wellness baseline.

MindSpace প্রোজেক্ট পরিচিতি

MindSpace একটি সম্পূর্ণ ক্লায়েন্ট-সাইড (Client-side) মুড ট্র্যাকিং অ্যাপ্লিকেশন, যা সিএসই (CSE) বিএসসি কোর্সের ওয়েব ডেভেলপমেন্ট প্রজেক্ট হিসেবে ডিজাইন করা হয়েছে।

🛠️ টেকনোলজি স্ট্যাক

কোনো এক্সটার্নাল লাইব্রেরি বা ফ্রেমওয়ার্ক ছাড়া শুধুমাত্র র-এইচটিএমএল (HTML5), সিএসএস (CSS3) এবং ভ্যানিলা জাভাস্ক্রিপ্ট (JavaScript) ব্যবহার করে এটি তৈরি।

💾 ডেটা প্রাইভেসি

এখানে HTML5 LocalStorage ব্যবহার করা হয়েছে, যার ফলে ইউজারের সমস্ত ডেটা তাদের নিজস্ব ব্রাউজারেই সুরক্ষিত থাকে এবং কোনো সার্ভারে আপলোড হয় না।

📊 ডাইনামিক অ্যানালিটিক্স

জাভাস্ক্রিপ্ট অ্যালগরিদমের মাধ্যমে রিয়েল-টাইমে ইউজারের গড় মুড স্কোর (Average Score) এবং মুড ডিস্ট্রিবিউশন চার্ট ক্যালকুলেট করা হয়।

💡 ব্যবহারকারী নির্দেশিকা: প্রতিদিন আপনার মানসিক অবস্থা অনুযায়ী ইমোজি সিলেক্ট করুন, প্রয়োজনে নোট লিখুন এবং "Log Mood" বাটনে ক্লিক করুন। পূর্বের রেকর্ড মুছে ফেলতে চাইলে "Clear All Data" বাটনটি ব্যবহার করুন।
